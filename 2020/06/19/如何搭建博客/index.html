如何搭建博客

发表于 2020-06-19 更新于 2020-06-24 阅读次数：
本文字数： 2k 阅读时长 ≈ 2 分钟

搭建博客可以选择wordpress、hexo等等，wordpress更简单。

本博客是由hexo加github仓库建立。

什么是hexo？

hexo

  • hexo 是一个生成静态html并且上传到服务器的工具。

  • 而github仓库就是托管博客页面的服务器

如何建立github仓库？

1、注册，具体方法网上很多不再累述

2、下载安装（默认安装即可）

3、绑定github

  • 利用Git的ssh服务创建公钥，利用公钥登录github，可以省去不必要的麻烦，避免每次登陆输入密码。

  • 打开gitbash输入ssh-keygen -t rsa命令，之后会在C:/Users/用户名/.ssh路径下生成id_rsa、id_rsa.pub（公钥），将公钥中内容复制下来。

  • 打开github，点击右上角头像，打开二级菜单中的setting选项。

  • 点击左侧SSH and GPG keys选项，选择右上角New SSH key，将复制的公钥内容粘贴到内容框，单击Add SSH key。

  • Git Bash 中输入 ssh -T git@github.com命令，查看是否出现successful提醒。

4、提交文件

  • 创建仓库，点击reposistor，选择new新建仓库，注意勾选初始化。

  • 克隆新建的仓库，点击Clone or download，复制地址，在准备建立本地仓库的文件夹右键打开git bash，输入git clone 地址（刚刚复制的），将远程仓库克隆到本地。

  • 在本地仓库文件夹内创建测试txt文件，在文件夹中右键打开gitbash，键入git status查看状态，之后会提示测试txt文件untracked，此时使用 git add 测试txt文件名 将文件添加到缓冲区，再使用git commit -m “commit text file”命令提交文件到本地仓库

  • 若提示输入用户名，键入两行代码

  • git status查看仓库状态，使用git push oringin master命令提交到github，之后会提示输入密码，登录github查看仓库文件是否上传成功。
